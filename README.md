# colav

Tools for extracting features from protein structures. 

`colav` provides methods for analyzing PDB files, either one-by-one or all together. The three methods currently implemented are: dihedral angles, pairwise distances, and strain. It is built on `biopandas` and Scientific Python software packages. 

## Installation 

You can install `colav` using `pip`: 

```
pip install colav
```
